# Keys configuration #

The game has a default keys configuration. Here it is (on an AZERTY keyboard) :

  * **Q** : Talk / push / pick up / throw
  * **W** : Attack
  * **X** : Inventory
  * **E** : Topic (disabled for now)
  * **Arrows** : Move
  * **Left Shift** : extra speed
  * **Tab** : See scores
  * **Escape** : Quit
